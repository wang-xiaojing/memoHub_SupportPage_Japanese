📋 List - メモ一覧表示モード

🎯 Listモードの概要

Listモードは作成済みのメモを一覧形式で表示し、メモの閲覧・編集・削除を行うための画面です。標準表示とLite表示の2つの表示モードがあり、用途に応じて切り替えができます。

📱 画面構成

ナビゲーションバー

  • + ボタン - 新しいメモの作成
  • Liteボタン - 表示モードの切り替え

メインリストエリア

  • メモアイテム - 各メモの詳細情報表示
  • スワイプアクション - 編集・削除操作
  • タップ操作 - メモの編集画面への移動

🔧 ツールバー機能

➕

新規作成ボタン

新しいメモを作成

📝

Liteボタン

表示モードの切り替え

新規作成ボタン（+ ボタン）

1Listモードから新規作成

  • 右上の「+」ボタンをタップ
  • 「New」モードに自動切り替え
  • 新しいメモの作成画面が表示

2検索結果リストから新規作成

  • 検索結果画面で「+」ボタンをタップ
  • 検索結果の最初のメモの日付を基準に設定
  • 「NewSpecifiedDate」モードで新規作成

👁️ 表示モード

標準表示モード（通常モード）

  • メモタイトル - 「Title」ラベル付きで表示
  • メモ日付 - 作成日時の表示
  • アラーム情報 - 設定されている場合はアイコン表示
  • テキストコンテンツ - メモ本文の一部を表示

Lite表示モード（簡易モード）

  • コンパクト表示 - 「Title」ラベルを非表示
  • メモ日付のみ - 基本情報のみ表示
  • テキスト非表示 - メモ本文は表示しない
  • スッキリした見た目 - より多くのメモを一覧で確認可能

Liteボタンの操作

1右上の「Lite」ボタンをタップ

2表示モードが切り替わります

  • 標準モード時 - ボタンが青色背景で「Lite」と表示
  • Liteモード時 - ボタンが白色背景・青色枠で「Lite」と表示
  • 設定は自動的にCore Dataに保存されます

📝 メモアイテムの表示内容

標準表示での情報

🏷️ タイトルエリア

  • 「Title」ラベル + メモタイトル
  • 作成日時（年月日 時分）
  • 右側にアラーム情報（設定時のみ）

📄 コンテンツエリア

  • テキストコンポーネントが有効な場合
  • 「Text」ラベル + メモ本文の一部
  • 長いテキストは自動的に省略表示

Lite表示での情報

📋 最小限の情報表示

  • メモタイトル（「Title」ラベルなし）
  • 作成日時
  • アラーム情報（設定時のみ）
  • テキスト本文は非表示

🔄 メモ操作方法

タップ操作

1メモアイテムをタップ

  • タップしたメモが編集用データにコピー
  • 「Edit」モードに自動切り替え
  • メモの編集画面が表示

スワイプアクション

1左にスワイプでアクションメニュー表示

🗑️

Delete

メモを完全削除

✏️

Edit

メモを編集

削除操作の注意

  • 削除は元に戻せません - メモ本体とすべての関連データが削除
  • Core Dataから完全削除 - データベースからも永久削除
  • 通知も削除 - 削除されたメモのIDが他の画面にも通知

🔔 アラーム表示機能

通知情報の表示

⏰ アラーム設定メモの識別

  • アラームが設定されたメモには専用アイコン表示
  • 通知の日時情報を自動取得
  • 期限切れの通知は自動的に除外

1通知検索機能

  • メモのIDと通知リストを照合
  • UNCalendarNotificationTrigger対応
  • UNTimeIntervalNotificationTrigger対応
  • 次回通知時刻の正確な表示

自動保存機能

  • 表示設定の記憶 - Liteモードの設定は自動保存
  • Core Data連携 - DisplayOptionエンティティで管理
  • アプリ再起動時も維持 - 設定は永続的に保存

🔍 空の状態表示

メモが存在しない場合

📭 空状態メッセージ

  • 「The specified memo items」
  • 「does not exist.」
  • グレーアウトした大きなテキストで表示
  • 新規作成を促すUI

🔄 データ同期と通知

リアルタイム更新機能

🔄 自動データ同期

  • 削除通知の受信 - 他画面での削除を即座に反映
  • NotificationCenter連携 - 「MemoItemDeleted」通知を監視
  • 画面表示時の初期化 - 通知リスナーの自動設定
  • 画面離脱時のクリーンアップ - リスナーの自動解除

💡 便利な使い方のコツ

効率的な操作方法

  • Liteモード活用 - 大量のメモを素早く確認したい時
  • 標準モード活用 - メモの内容も一緒に確認したい時
  • スワイプの活用 - 素早い編集・削除操作
  • 日付順ソート - 最新のメモが常に上部に表示

⚠️ 注意事項

  • 削除は慎重に - 削除操作は取り消しできません
  • 編集中の注意 - 他画面から戻る際は未保存データにご注意
  • 通知設定 - アラーム機能を使用する場合は通知許可が必要

🔗 関連機能

他の画面との連携

  • Newモード - +ボタンから新規作成画面へ
  • Editモード - タップ・スワイプから編集画面へ
  • 検索機能 - 検索結果もListモードで表示
  • カレンダー - 日付指定での新規作成連携
